# Interview Prep — Northwind Health · Staff Product Designer

## 1. Loop structure (fill in after recruiter screen)
- Recruiter screen (30 min) — positioning, level, remote policy, comp band. `VERIFY:` round mix.
- Hiring-manager screen (45 min) — role thesis, systems philosophy.
- Portfolio review (60–75 min) — D1 + D3.
- Whiteboard critique (45 min) — live.
- Take-home OR live design exercise (`VERIFY:` which).
- Cross-functional / values panel (45 min) — eng partner + PM.
- *No coding/algorithm round — this is a design loop.*

## 2. Study-areas table (gap-ranked; ~26 h total)
| Topic | Priority | Est. hours |
|---|---|---|
| D1 design-system case study — tighten to 60-sec + probing beats | P0 | 4 |
| Whiteboard-critique reps (structured critique, think-aloud) | P0 | 5 |
| Take-home/design-exercise dry run (audit → tokens → components) | P0 | 6 |
| Health-domain ramp (regulatory/a11y constraints — VERIFY specifics) | P1 | 3 |
| D3 data-informed dashboard case + the data-partner honesty line | P1 | 3 |
| Behavioral S1–S5 reps (esp. S2 influence, S3 conflict) | P1 | 3 |
| Northwind values re-read + "why Northwind" personal draft | P2 | 2 |

## 3. Set-pieces to rehearse (draw/do)
- **D1 — design-system architecture** (token layers → components → governance): draw it cold.
- **Whiteboard critique**: practice on 3 random SaaS screens; narrate goal → critique → systems fix.
- **Take-home shape**: time-boxed audit-and-propose; pre-build a reusable deliverable skeleton.

## 4. Gap areas (verbatim handling lines)
- *No manager title:* "I'm an IC multiplier — I mentor 3–4 designers and run our crit practice, and
  I'm specifically targeting Staff IC scope, not a management track."
- *New to health domain:* "Compliance constraints aren't new from fintech; I'd ramp on Northwind's
  specific regulatory and clinical constraints before touching the system."
- *Not a data scientist:* "I'm a data partner — I read funnels and co-design experiments; I don't
  build models."

## 5. Week-by-week plan
- **Week 1 — gaps + cases:** lock D1 and D3 walkthroughs; health-domain reading; draft "why Northwind."
- **Week 2 — set-pieces:** daily whiteboard-critique rep; full take-home dry run; review the systems
  token architecture until drawable cold.
- **Week 3 — behavioral + mocks:** S1–S5 reps; one mock portfolio review; one mock critique; refine
  the closer; logistics.
