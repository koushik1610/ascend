## Top 5 fixes (ranked by impact)

1. **Quantify the two flagship bullets.** The design system and the onboarding redesign are the whole
   case for Staff scope and currently carry no number. Pull from the metrics bank (M1–M6).
2. **Move keywords into achievement bullets.** ATS ranks contextualized keywords far above a skills
   list. "Design systems," "design tokens," "WCAG," "0→1" must live inside bullets, used once each.
3. **Add scope lines.** Each role needs a one-line scope (team size, org, surface) so a Staff screen
   can size the work.
4. **Resolve the title mismatch.** "Design Lead" (LinkedIn) vs. "Senior Designer" (resume) → "Senior
   Product Designer (IC)" everywhere. The "Lead" title is a true gap, not a claim.
5. **Flatten the format.** Remove the two-column skills sidebar and icon row; single column, standard
   section names ("Experience," "Skills," "Education").
